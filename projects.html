The computing innovation I chose to do my artifact on was A.I. but more specifically an A.I. that was recently researched and developed by the Swiss Federal Institute of Technology Lausanne (EPFL),thatâ€™s main function is to detect dead varro mites in beehives and record that data so that it can be used to let beekeepers know when to treat the respective beehives(Marc). My computational artifact explains the broad steps in the process of how the A.I. is used in detexting the Varro mites and relaying that information to the beekeeper.

The main form of data the beesaving A.I. uses is in the form of number of dead varro mites it detects within a hive. The beekeeper uploads the image of the hive (input) where the A.I. can then run through the picture over and over and attempt to pick up on all dead mites so it then can relay that information (output) to the beekeeper so the beekeeper knows exactly when to treat the hive. One main concern for this A.I. is storage which as it runs through larger and larger hives, it inevitably will run into too much data to handle which could lead to less and less accurate accounts of how many dead mites it detects which leads to the hive being mistreated.
